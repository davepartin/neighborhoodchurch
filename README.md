# Neighborhood Church Website

A clean, modern, and authentic church website built for Neighborhood Church in Overland Park, Kansas.

## What's Included

This package includes a complete, mobile-responsive website with:

- **Homepage** (index.html) - Hero, first visit info, Be/Bless/Belong framework
- **Sundays Page** (sundays.html) - Detailed service information and FAQ
- **Our Story Page** (about.html) - Church history, beliefs, and vision
- **Complete Styling** (style.css) - Modern design system with your brand colors
- **JavaScript** (script.js) - Mobile navigation and smooth scrolling

## Quick Start

1. **Replace Placeholder Images**
   - All images use Unsplash placeholders
   - Replace them with your actual church photos
   - Keep the same filenames or update the HTML accordingly

2. **Update Logo**
   - The site uses a simple house SVG icon
   - Replace with your actual NC logo (PNG format recommended)
   - Update the navigation and footer logo references

3. **Customize Content**
   - Review all text content and adjust to match your voice
   - Update any specific details (service times, programs, etc.)

4. **Create Missing Pages**
   - You'll need to create: community.html, leadership.html, give.html, connect.html
   - Follow the same structure as the existing pages

## Content Guidelines

When updating content, remember:
- Keep paragraphs conversational (3-5 sentences)
- Use "we" and "you" language
- Avoid churchy jargon unless you explain it
- Lead with benefits, not features
- Make calls-to-action clear and simple
